# NNS_Z ニコ生サーチ(雑)
![アプリスクリーンショット](https://github.com/TN8001/NNS_Z/blob/master/AppImage.png)
## 概要
ニコ生を５分おきに自動で再検索する専用ブラウザ？
## 特徴
* 非常にばかばかしい仕組みのわりに実用的（になった気がする
* ｃｓｓが得意であれば改造も自由にできる
* 次回検索までの待ち時間確認アニメーション付き
## 使い方
1. 起動する
1. 検索キーワードを入れ検索ボタンを押す
1. 気になる放送をクリックする（規定ブラウザで開きます
1. （５分おきに自動で再検索されます）
